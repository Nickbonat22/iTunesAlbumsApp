# iTunesAlbumsApp
An iOS app that shows the top 100 albums on iTunes using: https://rss.itunes.apple.com/en-us

Made with: 
* Swift
* Auto Layout
* No storyboards or nibs
* No third party libraries
* No force unwraps
* Unit testing
